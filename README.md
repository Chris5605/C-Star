# C-Star
A console based C-language noise generator.

Development date goal: 6/1 - 6/30. 

This is my first public project, intended to help me work on my C development. Hopefully it will be the start of series of monthly projects. Check back later for progress. 

Goals: 
- Easy to use CLI interface
- Lightweight runtime for potential background usage
- Live audio generation of various colors of noise (white, pink, gray, etc.)
- User specification for various tonal controls
- Export to MP3 or other audio format if time
